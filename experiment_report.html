评测指标说明

Bleu-1 / 2 / 3 / 4
Bilingual Evaluation Understudy
衡量生成句子与参考句子之间 n-gram 短语重叠的精确率。 Bleu-1 看单词匹配，Bleu-4 看连续 4 个词的片段匹配。 数值越高说明生成文本与人工标注在字面上越接近。
词汇 n-gram 匹配
METEOR
Metric for Evaluation of Translation with Explicit ORdering
在 Bleu 基础上引入词干还原、同义词匹配，并同时考虑精确率与召回率的调和均值（F-score）。 对语义近似但用词不同的表达更宽容，与人类判断的相关性优于 Bleu。
n-gram + 语义扩展
ROUGE-L
Recall-Oriented Understudy for Gisting Evaluation
基于最长公共子序列（LCS），衡量生成句与参考句中保留的最长有序词序列比例。 对句子整体流畅度和结构连贯性更敏感。
最长公共子序列
CIDEr
Consensus-based Image Description Evaluation
专为图像描述任务设计。用 TF-IDF 加权 n-gram，奖励罕见但精准的词汇，惩罚过于通用的描述（如"一张图"）。 与人类对图像描述质量的判断相关性最高，是当前图像描述的主评指标。
图像描述主指标
SPICE
Semantic Propositional Image Caption Evaluation
将描述解析为语义场景图（对象、属性、关系），再比较生成描述与参考描述的场景图 F-score。 对物体识别、属性描述、空间关系等语义理解能力最为敏感，但对词汇选择不敏感。
场景图语义匹配

关键结论

  • 1
    三强并列 —— SigLIP2（0.9333）、RADIO（0.9332）、EVA-ViT（0.9306）在当前训练规模下实质性能相当，均超过基线水平，差距处于统计噪声范围内。
  • 2
    RADIO 的 Q-Former 对齐能力最强 —— Stage 1 的 loss_itc = 0.898，低于其他所有编码器。C-RADIOv4 同时蒸馏了 CLIP 和 DINOv3 双教师，视觉特征天然携带较强的语言语义信号。
  • 3
    DINOv3 差距属于架构本征限制 —— DINOv3-ViT-L/16 使用 patch_size=16（每张图 token 数少于 patch_size=14 的模型），且仅进行自监督预训练，不含语言对齐目标。bug 修复后 CIDEr 提升至 0.8547，但与顶部模型仍有明显差距，反映其在图像描述任务上的固有能力天花板。
  • 4
    输入归一化必须与编码器匹配 —— SigLIP2 要求 mean=std=0.5；RADIO 内部自带 InputConditioner 处理归一化，外部应传入原始 [0,1] 像素值（do_normalize=false）；EVA-ViT 和 DINOv3 使用标准 CLIP/ImageNet 归一化。归一化不匹配会导致严重的性能退化。
